Código fonte

O comando document.getElementById('teste') é escrito em linguagem JavaScript.

o comando code, coloca o texto na forma mono-espaçada, serve para indicar um código principalmente.

exemplo de código em linguagem python:

num = int(input('Digite um número: ')) if num % 2 == 0: print('esse número é par!') else: print('esse número é ímpar')

Observando o código acima, percebe-se que não há indentação. Isso acontece porque a tag code apenas deixa a fonte no formato de código, para termos a indentação, envolvemos a tag code com nosso código dentro na tag pre:

        
num = int(input('Digite um número: '))
if num % 2 == 0:
    print('esse número é par!')
else:
    print('esse número é ímpar')
