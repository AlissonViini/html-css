O h1 é um exemplo de caixas box-level

Parágrafos são também um exemplo de caixa block level, mas os links são exemplos de caixas inline level

Já esse link que era inline-level, se tornou box-level, bor conta do display block, é possível também transformar elementos box em inline level usando o display inline. E comando inline-block, transforma um block em inline, sendo que podrão ser alteradas algumas características como largura e altura
